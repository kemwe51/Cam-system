# Android usage

The web app is a mobile-first PWA.

## Local use

1. Run the API on port `3001`.
2. Run the web app and open it in Chrome on Android.
3. Use **Add to Home Screen** to install the PWA.

## Mobile behavior

- Cards stack vertically on narrow screens.
- Primary actions stay grouped near the top for thumb reach.
- Lists use compact cards so features, operations, and risks remain reviewable on phones.

## Current limitations

- Offline support is limited to a very small cached shell.
- Authentication and multi-user approval history are not yet implemented.
- The web app currently uses the sample JSON flow rather than file ingestion.
